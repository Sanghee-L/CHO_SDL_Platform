# Synthetic Dataset

## Overview

This directory contains the synthetic datasets used to develop and evaluate the CHO SDL Platform V1 prototype.

The datasets were generated to simulate realistic CHO cell line development (CLD) workflows while avoiding the use of proprietary or experimental manufacturing data.

The synthetic data supports development of:

- Clone screening algorithms
- Multi-objective optimization
- Digital twin simulation
- Reinforcement learning workflows
- Host cell engineering simulations
- SDL feedback systems
- Executive clone intelligence dashboards

---

## Directory Structure

### raw/

Synthetic source datasets used throughout the platform.

Files:

- cld_5000clones_legacy.db
  - SQLite database containing 5,000 synthetic CHO clones.

- batch_effects_truths_5000_legacy.csv
  - Simulated batch-to-batch variation parameters.

- clone_latent_truths_5000_legacy.csv
  - Hidden biological state variables used for simulation and validation.

---

### outputs/

Final outputs generated by the SDL workflow.

Files:

- notebook23_sdl_learning_database_5000_legacy.csv
  - SDL learning history and knowledge update records.

- notebook24_clone_intelligence_platform_5000_legacy.csv
  - Executive clone intelligence output.

- notebook24_executive_scorecard_5000_legacy.csv
  - Final SDL scoring results.

- notebook24_top_manufacturing_candidates_5000_legacy.csv
  - Recommended manufacturing candidates.

---

## Disclaimer

All datasets are synthetic and were generated for research, educational, and software development purposes.

The datasets do not represent real experimental results and should not be used for manufacturing or development decisions.
